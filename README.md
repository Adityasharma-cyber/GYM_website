# Robur Fitness Frontend

## About the Project

Robur Fitness Frontend is a beginner-friendly gym website built using HTML, CSS, and JavaScript. This project was created to practice frontend web development and improve my understanding of website layout, styling, and basic interactivity.

Note: This repository contains only the frontend of the website. No backend, database, or authentication system has been implemented. The current version is designed primarily for desktop screens and is not fully responsive for mobile devices.

## Features

Clean and simple user interface.

Desktop-focused layout.

Navigation bar.

Hero section.

Membership and pricing section.

Contact section and footer.

Easy-to-understand code structure for beginners.

## Technologies Used

HTML5 - Structure of the website.

CSS3 - Styling and layout.

## How to Run

Download or clone this repository.

Open the project folder in VS Code or any code editor.

Open index.html in your web browser.

## Learning Purpose

This project was built as a beginner frontend project to practice HTML, CSS, JavaScript, and GitHub.
